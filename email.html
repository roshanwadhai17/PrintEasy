## Troubleshooting ### Common Issues: 1. **"EmailJS is not defined" Error** - Make sure you're connected to the internet - The EmailJS script loads automatically 2. **Email Not Sending** - Double-check all IDs are correct - Verify your email service is connected - Check EmailJS dashboard for error logs 3. **Email Goes to Spam** - Add your sending email to contacts - Check spam/junk folder - Use a verified domain for production ### EmailJS Limits: - **Free Plan:** 200 emails/month - **Personal Plan:** 1,000 emails/month ($15) - **Professional Plan:** 10,000 emails/month ($70) ## Advanced Features (Optional) ### Auto-Reply Setup: - Create a separate template for order confirmations - Send immediate confirmation when order is placed ### Attachment Support: - EmailJS supports attachments - Can send order receipts or shop location maps ### Multiple Languages: - Create templates in Hindi, Marathi for local customers - Dynamic template selection based on customer preference ## Security Notes - Never share your Private Key - Use only the Public Key in frontend code - Monitor your usage in EmailJS dashboard - Set up email quotas to prevent abuse --- **üéâ Congratulations!** Your Xerox shop now has professional email notifications that will delight your customers and streamline your business operations.
